# TM FIT Platform

Prima versione verificata della piattaforma TM FIT in stile luxury medical.

## File principali

- `app/page.js` - WebApp principale
- `app/layout.js` - Layout Next.js
- `app/globals.css` - Stili globali Tailwind
- `lib/demoData.js` - Dati demo clienti, misurazioni e allenamenti
- `supabase_schema.sql` - Schema iniziale futuro database

## Deploy su Vercel

1. Carica il contenuto di questa cartella nel repository GitHub `tmfit-platform`
2. Torna su Vercel
3. Importa il repository
4. Premi Deploy

## Stato

Questa versione è una MVP demo frontend. Il login è dimostrativo.
Il prossimo step sarà collegare Supabase per dati reali e autenticazione.
