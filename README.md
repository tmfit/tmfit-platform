# TM FIT Platform MVP

Prima versione della piattaforma TM FIT in stile luxury medical.

## Avvio locale
```bash
npm install
npm run dev
```

Poi apri `http://localhost:3000`.

## Deploy su Vercel
1. Crea repository GitHub `tmfit-platform`
2. Carica questi file
3. Su Vercel fai Import Git Repository
4. Premi Deploy

## Prossimo step
Collegamento Supabase per login reale, clienti, visite, allenamenti e PDF.
